# Login Form

A clean, responsive login form built with plain HTML, CSS and a touch of JavaScript.

![Login Form Screenshot](https://github.com/BinaryVortex/Login-Form-122/blob/main/Screenshot%202025-01-31%20012314.png)

---

## About

This repository contains a small, modern-looking login form UI intended as a frontend template or learning example. It demonstrates layout, styling, and simple client-side validation using HTML, CSS, and JavaScript.

## Demo

To view the form locally:

1. Clone the repo:

   git clone https://github.com/BinaryVortex/Login-Form-122.git
2. Open `index.html` in your browser (double-click or use `File -> Open`)

There is no backend included; the form is for UI/demo purposes only.

## Features

- Responsive layout
- Smooth, modern styling
- Basic client-side validation (JavaScript)
- Small, easy-to-read codebase — great for learning or quick prototyping

## Files

- `index.html` — markup for the login form
- `style.css` — styling and responsive rules

## Technologies

- HTML
- CSS
- JavaScript

## Customization

- Colors: edit variables or color rules in `style.css`.
- Inputs: add/remove fields in `index.html`.
- Validation: extend `script` in the HTML or move to a separate `.js` file.
